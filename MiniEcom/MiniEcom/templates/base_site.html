MiniEcom Admin Dashboard

MiniEcom is a demonstration e‑commerce platform built with Django. It highlights the essential features of an online store, including product listings, customer management, and order tracking. This project showcases how a cloud‑hosted Django application can integrate static and media files, provide a secure admin interface, and deliver a responsive Bootstrap‑styled homepage. The admin panel serves as the control center where administrators can manage inventory, monitor customer activity, and ensure smooth operation of the store.
